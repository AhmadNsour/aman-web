# Aman Web

Aman Web is the public website and driver onboarding portal for Aman, a women-only ride-hailing platform in Jordan.

The project includes:

- Home page
- About page
- Support page
- Become Driver registration flow
- Shared navigation layout
- Responsive UI
- Tailwind CSS styling
- shadcn/ui components
- Lucide icons

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide React

## Getting Started

Install dependencies:

```bash
npm install
```
